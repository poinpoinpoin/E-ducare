HTML

HTML, que significa "HyperText Markup Language" (Lenguaje de Marcado de Hipertexto), es el lenguaje estándar utilizado para crear la estructura y el contenido de las páginas web. Permite definir la forma en que se presentan los elementos en una página y cómo se relacionan entre sí.

A continuación, te mostraré los elementos esenciales de HTML que necesitas conocer:

Estructura basica: 

<!DOCTYPE html>
<html>

<head>
   <title>Título de la presentacion</title>
</head>

<body>
   Contenido de la página
</body>

</html>
  • <!DOCTYPE html>: Declaración que indica que el documento sigue el estándar HTML5.
  • <html>: Elemento raíz que envuelve todo el contenido de la página.
  • <head>: Encabezado de la página que contiene información meta y enlaces a archivos externos.
  • <title>: Define el título de la página que se muestra en la pestaña del navegador.
  • <body>: Contenedor principal del contenido visible en la página.

Encabezados

Los encabezados se utilizan para definir los títulos y subtítulos de una página. Van desde "<h1>" hasta "<h6>", donde "<h1>" es el más grande y "<h6>" el más pequeño. 

<h1>Título principal</h1>
<h2>Subtítulo</h2>

Párrafos:

Los párrafos se crean utilizando el elemento "<p>". 

<p>Este es un párrafo de ejemplo.</p>

Enlaces:

Los enlaces se crean utilizando el elemento "<a>". El atributo "href" define la URL a la que se enlaza. 

<a href="https://www.ejemplo.com">Enlace de ejemplo</a>

Imágenes:

Las imágenes se insertan utilizando el elemento "<img>". El atributo "src" especifica la URL de la imagen. 

<img src="imagen.jpg" alt="Descripción de la imagen">

Listas:

Puedes crear listas ordenadas "<ol>" o listas sin orden "<ul>". Los elementos de la lista se definen con "<li>". 

<ul>
   <li>Elemento 1</li>
   <li>Elemento 2</li>
</ul>

Estos son solo algunos elementos básicos de HTML. A medida que te familiarices con ellos, puedes explorar elementos más avanzados como tablas, formularios, estilos CSS y mucho más.

Recuerda que HTML define la estructura y el contenido de una página web, pero para aplicar estilos y presentación visual, necesitarás utilizar CSS (Cascading Style Sheets). Juntos, HTML y CSS forman la base de desarrollo web.

¡Espero que este curso introductorio sea útil para comenzar a aprender HTML!
