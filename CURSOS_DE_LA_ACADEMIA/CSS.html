CSS

CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para describir la presentación y el diseño de un documento HTML (o XML). Se utiliza para controlar el aspecto visual de los elementos en una página web, como el color, la tipografía, el espacio, el diseño y otros aspectos visuales.

Sintaxis básica de CSS:

Una regla CSS consta de un selector, una propiedad y un valor. La sintaxis general es la siguiente: 

selector {
   propiedad: valor;
}
  • Selector: Identifica el elemento HTML al que se utilizará el estilo. Puede ser una etiqueta HTML, una clase, un ID o un selector de atributo.
  • Propiedad: define qué característica del elemento se desea estilizar, como el color del texto o el tamaño de la fuente.
  • Valor: Especifica el valor que se asignará a la propiedad, como "rojo" para el color del texto o "14px" para el tamaño de fuente.

Métodos para aplicar estilos CSS:

  • Estilos en línea: Se aplica directamente en el atributo "<style>" de un elemento HTML específico. Por ejemplo: 
    • <h1 style="color: blue; font-size: 20px;">Título</h1>
  • Estilos incrustados: Se define dentro de la sección "<style>" en la cabecera del documento HTML. Los estilos se aplican a los elementos seleccionados en todo el documento. Por ejemplo: 
    • <head>
       <style>
           h1 {
               color: blue;
               font-size: 20px;
           }
       </style>
    </head>

    <body>
       <h1>Título</h1>
    </body>
  • Estilos externos: Se define en un archivo CSS separado y se enlazan al documento HTML utilizando la etiqueta "<link>". El archivo CSS contiene todas las reglas de estilo para el sitio web y se puede reutilizar en múltiples páginas HTML. Por ejemplo: 
    • <head>
       <link rel="stylesheet" href="estilos.css">
    </head>

    <body>
       <h1>Título</h1>
    </body>

Selectores y Propiedades Básicas

En CSS, los selectores se utilizan para seleccionar elementos HTML y aplicarles estilos. Las propiedades definen cómo se verán esos elementos. Veamos algunos ejemplos: 

/* Selector de etiqueta */
p {
   color: blue;
}

/* Selector de clase */
.mensaje {
   font-size: 18px
}

/* Selector de ID */
#titulo {
   text-align: center;
}

/* Selector descendiente */
.contenedor div {
   background-color: gray;
}

Modelo de Caja y Posicionamiento

El modelo de caja de CSS define cómo se representa un elemento HTML. Puede controlar el tamaño, el margen, el relleno y la posición de un elemento. Veamos algunos ejemplos: 

/* Cambiar el tamaño de la caja */
.mi-elemento {
   width: 200px;
   height: 100px;
}

/* Agregar margen y relleno */
   margin: 10px;
   padding: 20px;
}

/* Controlar el posicionamiento */
   position: relative;
   top: 50px;
   left: 20px;
}

Diseño Responsivo y Media Queries

CSS también te permite crear diseños responsivos para diferentes dispositivos. Puedes utilizar media queries para aplicar estilos específicos según el tamaño de la pantalla. Veamos un ejemplo: 

/* Estilo para pantallas pequeñas */
@media (max-width: 600px) {
   .mi-elemento {
       font-size: 14px;
   }
}

Animaciones y Transiciones

Puedes crear animaciones y transiciones suaves utilizando CSS. Veamos un ejemplo basico: 

/* Animación básica */
@keyframes mi-animation {
   0% { opacity: 0; }
   100% { opacity: 1; }
}

.mi-elemento {
   animation: mi-animation 2s infinite;
}

/* Transición al pasar el cursor */
.mi-elemento {
   transition: background-color 0.3s ease;
}

.mi-elemento:hover {
   background-color: yellow;
}

Estas son solo algunas de las muchas características de CSS. A medida que avances, podrá explorar más propiedades y técnicas para crear estilos y diseños avanzados.
