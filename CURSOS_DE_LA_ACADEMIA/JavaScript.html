JavaScript

JavaScript es un lenguaje de programación utilizado principalmente para crear interactividad en sitios web. Puede ejecutarse en el lado del cliente (navegador web) y en el lado del servidor (Node.js). Aquí hay algunos conceptos clave que debes conocer:

Variables:

En JavaScript, las variables se utilizan para almacenar datos. Puedes declarar una variable usando la palabra clave "var", "let" o "const". Por ejemplo: 

var mensaje = "Hola, mundo!";
let numero = 10;
const PI = 3.14;

Tipos de datos:

JavaScript es un lenguaje de tipado dinámico, lo que significa que no es necesario declarar claramente el tipo de una variable. Los tipos de datos comunes incluyen cadenas de texto, números, booleanos, arreglos y objetos.

Operadores:

JavaScript admite una variedad de operadores aritméticos (+, -, *, /), operadores de comparación (==, !=, <,>), operadores lógicos (&&, ||, !) y más.

Estructuras de control:

Puedes controlar el flujo de ejecución en JavaScript utilizando estructuras como condicionales "if-else" y bucles "for" y "while". Aquí tienes un ejemplo: 

if (edad >= 18) {
   console.log("Eres mayor de edad");
} else {
   console.log("Eres menor de edad");
}


for (let i = 0; i < 5; i++) {
   console.log(i);
}


let i = 0;
while (i < 5) {
   console.log(i);
   i++
}

Funciones:

Las funciones te permiten agrupar un conjunto de instrucciones y ejecutarlas cuando sea necesario. Puedes definir una función de la siguiente manera: 

function saludar(nombre) {
   console.log("Hola, " + nombre + "!");
}


saludar("Juan");

Manipulación del DOM El Document Object Model (DOM) representa la estructura de un documento HTML y proporciona una interfaz para interactuar con él. En JavaScript, puedes acceder y manipular elementos HTML utilizando el DOM. Aquí hay algunos ejemplos:

Acceder a elementos:

Puede seleccionar elementos por su ID, clase, etiqueta, etc. utilizando las funciones proporcionadas por el DOM. Por ejemplo: 

let elemento = document.getElementById("miElemento");
let elementos = document.getElementsByClassName("miClase");
let etiquetas = document.getElementsByTagName("p");

Manipulación de elementos:

Puedes cambiar el contenido HTML, los estilos y otros atributos de un elemento. Aquí tienes un ejemplo: 

elemento.innerHTML = "Nuevo contenido";
elemento.style.color = "red";
elemento.setAttribute("class", "nuevaClase");

Eventos:

JavaScript te permite responder a eventos como clics de botón, movimientos del mouse, etc. Puedes asignar funciones a los eventos de los elementos. Por ejemplo: 

elemento.addEventListener("click", function() {
   console.log("¡Haz hecho clic en el elemento!");
});

AJAX y solicitudes HTTP JavaScript te realiza solicitudes HTTP asíncronas permite utilizar la técnica de AJAX. Esto te permite cargar datos desde un servidor y actualizar partes específicas de una página web sin tener que recargarla por completo. Aquí hay un ejemplo básico: 

let xhttp = new XMLHttpRequest();
xhttp.onreadystatechange = function() {
   if (this.readyState === 4 && this.status === 200) {
     console.log(this.responseText);
   }
};
xhttp.open("GET", "https://api.example.com/data", true);
xhttp.send();
