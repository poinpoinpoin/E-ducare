React

React es una biblioteca de JavaScript desarrollada por Facebook que se utiliza para construir interfaces de usuario interactivas y reutilizables. React utiliza un enfoque basado en componentes, lo que significa que puedes crear componentes de UI independientes y componerlos para construir aplicaciones completas.

Configuración del entorno de desarrollo

Antes de comenzar a desarrollar con React, debe configurar su entorno de desarrollo. A continuación, se presentan los pasos básicos para configurar un proyecto de React:

  • Instala Node.js en tu computadora https://nodejs.org/. Node.js incluye npm, que es el administrador de paquetes que necesita para instalar las dependencias de React.
  • Crea un nuevo proyecto de React utilizando Create React App (CRA). CRA es una herramienta oficial de Facebook que te ayuda a configurar rápidamente un proyecto de React con todas las dependencias necesarias. Para crear un nuevo proyecto de React, abre tu terminal y ejecuta el siguiente comando: 
    npx create-react-app mi-aplicacion-react
    Esto creará un nuevo directorio llamado "mi-aplicacion-react" con una estructura de proyecto básico.
  • Navega al directorio de tu nuevo proyecto: 
    cd mi-aplicacion-react
  • Inicia el servidor de desarrollo 
    npm start
    Esto iniciará el servidor de desarrollo de React y podrá ver su aplicación en el navegador visitando http://localhost:3000 .

Componentes y JSX

En React, los componentes son la base de la construcción de interfaces de usuario. Un componente representa una parte reutilizable y modular de la interfaz de usuario. Puedes crear componentes funcionales o componentes de clase en React.

React utiliza JSX (JavaScript XML) para definir la estructura de los componentes y sus elementos. JSX te permite mezclar código JavaScript con marcado HTML-like. Aquí hay un ejemplo básico de un componente funcional en React: 

import React from 'react';

import MiComponente() {
   return {
       <div>
           <h1>Hola, mundo!</h1>
       </div>
   );
}
import default MiComponente;

Props y State

Los componentes pueden recibir datos a través de props y tener un estado interno llamado state.

Las props son datos que se pasan de un componente padre a un componente hijo. Puedes usar las props para personalizar y configurar un componente.

El estado es un objeto especial que contiene datos que pueden cambiar a lo largo del tiempo y afectar la representación de los componentes. Puedes usar el estado para mantener información dinámica en tu aplicación.

Aquí hay un ejemplo de cómo utilizar props y state en un componente de clase en React: 

import React, { Component } from 'react';

class MiComponente extends Component {
   constructor(props) {
       super(props);
       this.state = {
           contador: 0,
       };
   }

   handleClick() {
       this.setState({ contador: this.state.contador + 1 });
   }
   render() {
       return (
           <div>
               <h1>{this.props.titulo}</h1>
               <p>Contador: {this.state.contador}</p>
               <buttom onClick={() => this.handleClick()}>Incrementar</buttom>
           </div>
       );
   }
}

export default MiComponente;

Ciclo de vida de los componentes

Los componentes de clase en React tienen un ciclo de vida que te permite ejecutar un código en momentos específicos durante la existencia del componente, como cuando se monta, actualiza o desmonta.

Algunos métodos del ciclo de vida común son:

  • componentDidMount(): se llama después de que el componente se monta en el DOM.
  • componentDidUpdate(prevProps, prevState): se llama después de que el componente se actualice.
  • componentWillUnmount(): se llama justo antes de que el componente se desmonte y se elimine del DOM.

Puedes utilizar estos métodos para realizar tareas como llamar a una API, suscribirte a eventos o limpiar recursos cuando el componente se desmonte.

Estos son solo los conceptos básicos para comenzar con React. A medida que avances, puede aprender sobre enrutamiento, gestión de estado avanzado (como Redux o Context API), animaciones y más.
